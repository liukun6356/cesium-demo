创建Map

框架的核心静态方法，它用来在页面中创建地图.

使用示例

                    
das3d.createMap({
    id: 'cesiumContainer',
    url: "config.json",
    success: function (viewer, jsondata) {//地图成功加载完成后执行

    }
});
